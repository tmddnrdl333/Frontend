# 03. 개발환경 설정, 프로젝트 진행



## 작업환경 설정하기

### NODE / NPM 

Node.js는 자바스크립트를 서버환경에서 실행하는 자바스크립트 런타임.

> 런타임 : 특정 언어로 만든 프로그램들을 실행할 수 있는 환경을 말함.

NPM은 Node.js로 만들어진 모듈들을 설치 및 버전관리 할 수 있는 프로그램.
